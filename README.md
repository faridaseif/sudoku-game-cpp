# 🧩 Sudoku Game (C++ GUI)

A clean, interactive 9x9 Sudoku puzzle built with C++ and Windows Forms. This project demonstrates Object-Oriented Programming (OOP) principles and graphical event handling.

## 🎮 Features
* **Interactive Grid:** Responsive 9x9 Sudoku interface.
* **Real-time Validation:** Instant checking for Sudoku rule violations.
* **Game Logic:** Pre-generated boards with win condition detection.
* **Clean GUI:** Minimalist design focused on user experience.

---

## ⚙️ Tech Stack
* **Language:** C++ / CLI
* **Framework:** .NET / Windows Forms (GUI)
* **Architecture:** Object-Oriented Programming (OOP)
* **IDE:** Visual Studio 2022

## **🚀 How to Run**
```text
1. Clone the Repository
Bash
git clone [https://github.com/faridaseif/sudoku-game-cpp.git](https://github.com/faridaseif/sudoku-game-cpp.git)
2. Open in IDE
Launch Visual Studio.

Open the .sln (Solution) file or the project folder.

3. Build & Requirements
Ensure the "C++ desktop development" workload is installed in your Visual Studio Installer.

Make sure the project is set to x86 or x64 to match your system.

4. Run
Press F5 (or click the Start button) to compile and launch the game.
---
```
